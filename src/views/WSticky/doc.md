#### 使用

| 参数名      | 作用         | 默认值               | 是否必须 |
| ----------- | ------------ | -------------------- | -------- |
| stickyTop | 距离顶部距离 | 0                 | no       |
| zIndex | 层级            | 1                 | no       |
| className | 类名            | 空                 | no       |

  <div>
    <WSticky></WSticky>
  </div>
</template>

    }
